## Prerequisites

The course aims to be accessible to **students without prior experience in programming**. Students are expected to have some prior experience or interest in geospatial or environmental science topics.

## Assessment

This unit is assessed through an end-of-unit report with a length of 2,500 words. Your work for this report will involve using different scientific computing tools (e.g. GitHub and version control, NumPy and basic programming with Python) in geospatial applications. The report will be written in a reproducible manner and will include scientific graphs and tables with the necessary code.
